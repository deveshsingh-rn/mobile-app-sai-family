# Sai Family Mobile

Expo React Native app for iOS, Android, and web.

## Commands

```bash
npm install
npm run start
npm run android
npm run ios
npm run web
npm run lint
```

From the repo root, use:

```bash
npm run mobile:start
npm run mobile:android
npm run mobile:ios
npm run mobile:web
```

The mobile app is intentionally separate from `backend-api` so mobile builds and backend development can be owned, deployed, and scaled independently.
